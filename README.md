# Eric Bragas - Personal Portfolio Website

## Overview
This is a personal portfolio website showcasing my professional work, skills, and contact information.

## Features
- Responsive, modern design
- Smooth scrolling navigation
- Project showcase
- Contact form
- Animated sections

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome for icons

## Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. No additional setup required

## Customization
- Update content in `index.html`
- Modify styles in `styles.css`
- Adjust interactions in `script.js`

## Deployment
This website can be easily deployed to:
- GitHub Pages
- Netlify
- Vercel
